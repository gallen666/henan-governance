# 河南省高效能治理交互模拟系统

基于 MIT En-ROADS 方法论 + 史文中教授 ISUI 智慧城市指数体系的交互式治理推演平台。

## 功能模块（10 大栏目）

- 🏠 首页 · 总览
- 🎛️ 政策沙盘 · 18 项杠杆 + 系统动力学
- 📐 方法论 · En-ROADS 借鉴
- 📚 论文拆解 · 11 章博士论文
- 🗂️ 案例库 · 郑州模式
- 📋 调查问卷 · 15 题实测
- 🩺 AI 诊断 · 4 维分析
- 🌆 智慧城市指数 · ISUI 6 维
- 🎯 工作坊指南
- ℹ️ 关于

- 《河南省基层群众满意度指数体系》
