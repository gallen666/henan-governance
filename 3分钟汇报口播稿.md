## 📹 屏幕录制操作（推荐路径）

### 方案 A：本人投屏现场汇报（推荐）
1. 双击打开 `3分钟视频演示.html`
2. 按 **F11 全屏**
3. 投屏到会议室大屏
4. 点击中央播放按钮 → 您本人按本稿口播
5. 视频底部字幕作为辅助提示，关闭可按"字幕 OFF"

### 方案 B：录制成 MP4 视频文件
**Mac**：
1. 打开 `3分钟视频演示.html` 全屏播放
2. 按 `⌘ + Shift + 5` → 选择"录制选定窗口"或"录制全屏"
3. 同步用 QuickTime 或外部话筒录入您的口播
4. 导出 .mov，可用 HandBrake 转 MP4

**Windows**：
1. 全屏打开页面
2. 按 `Win + G` 调出 Xbox Game Bar
3. 点录制按钮，同步配音
4. 导出 MP4

**专业工具（推荐高质量产出）**：
- **OBS Studio**（免费，Mac/Win/Linux 通用）
- **Loom**（云端剪辑，便于分享链接）
- **Camtasia**（含字幕、配音、剪辑一体）

### 方案 C：先录制画面 + 后期配音
1. 用 OBS 录制视频页面静音版（关字幕）
2. 在 Adobe Premiere / 剪映 中导入
3. 按本稿录制纯音频，对齐时间码
4. 加上配乐 + 字幕条 → 导出最终 MP4

1. **开场前留 3 秒静默**——让领导从手头事务转移注意力

4. **场景 7 结尾后停顿 5 秒**——让金句沉淀
5. **结束后立刻递上备忘录**——一页纸总结 + 一个二维码（指向完整网站）
